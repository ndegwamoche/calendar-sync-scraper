- **Key Value Propositions:**
  - Highly reliable and efficient synchronization engine
  - Supports multiple calendar formats and platforms
  - Easy-to-use API for integrating into existing applications

### Configuration Options

```json
{
  "calendarServices": [
    {
      "name": "Google Calendar",
      "apiKey": "your-google-calendar-api-key"
    },
    {
      "name": "Outlook Calendar",
      "apiKey": "your-outlook-calendar-api-key"
    }
  ]
}
```

### API Documentation

#### Sync Calendars

```http
POST /api/sync
Content-Type: application/json

{
  "source": "google",
  "target": "outlook"
}
```
